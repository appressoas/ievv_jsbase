    ScriptLoaderSingleton

    Asyncronous script loader.

    Example:

    const scriptLoader = new ScriptLoaderSingleton();
scriptLoader.load('//example.com/myscript.js').then((script) => {
  console.log(`The ${script.src} script was loaded`);
});
